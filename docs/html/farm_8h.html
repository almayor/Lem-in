int run_farm ( t_farm *  farm,
t_list **  paths,
t_map *  vertex_names 
)

Run ants in the farm along the given paths.

For each ant, allocates a path and runs it along it. If several ants share a path (in cases where there are fewer paths than ants), wait until the ant in front has moved ahead before putting the next ant on this path at the start. Print out each ant's position at every step, given a map from vertex id's to vertex names;

Parameters
farmThe ant farm
pathsAn NULL-terminated array with all existing paths between the source and the sink, where each path is a list of vertex id's
vertex_namesA map between vertex id's and vertex names
Returns
0 on success, 1 on error
Exceptions
EINVALNo single path exists, or paths are invalid.
