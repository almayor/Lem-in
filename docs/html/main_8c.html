Reads links and returns paths found by the Edmonds-Karp algorithm.

Reads the edges (links) from STDIN, writes them to a graph, then uses the Edmonds-Karp algorithm to get paths from the source to the sink. The returned paths are sorted in the order from shortest to longest.

Note
This function reprints what it read from STDIN to STDOUT.
Returns
An ordered NULL-terminated array of paths from the source to the sink, or NULL if an error occurs
Parameters
n_antsNumber of ants and, therefore, the maximum number of paths that must be considered
vertex_namesA map between vertex id's and vertex names
Exceptions
ENOMEMMemory allocation error
EINVALThe format is wrong, the edges connect vertices that do not exist, or the source and the sink aren't connected
