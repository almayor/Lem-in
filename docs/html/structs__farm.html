◆ ants

s_farm::ants

A list of position on each ant's remaining path.

Example: If the first ant's path is 1->2->3->4 and the ant is currently at vertex 2, then ants[0] points at the path's link holding the vertex 2. Please note that several ants' positions can point to different links in the same linked list. Therefore, as ants progress along their paths no memory should be freed or allocated.
