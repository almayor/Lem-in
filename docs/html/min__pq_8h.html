◆ t_min_pq

typedef struct s_min_pq t_min_pq

A priority queue of fixed size.

It supports the usual insert and delete-the-minimum operations as well as change-the-value methods. Each value in the queue is associated with an integer handle.

Note
Implemented as a binary heap along with arrays to associate values with integers in the given range.
